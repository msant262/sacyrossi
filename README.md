# 🎮 Sacy Rossi - Site Oficial

Site oficial do ex-pro player de Valorant e atual streamer da MIBR, Sacy Rossi. Campeão do Masters 2022 de Valorant e primeiro atleta campeão de Masters e Champions no Valorant.

## 🌟 Sobre o Projeto

Este é um site moderno e responsivo desenvolvido para apresentar a carreira e conquistas do Sacy Rossi, um dos jogadores mais respeitados do cenário brasileiro de Valorant. O site destaca sua jornada desde o Counter-Strike aos 11 anos até se tornar campeão mundial no Valorant e streamer oficial da MIBR.

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone [url-do-repositorio]
cd sacyrossi
```

2. Abra o arquivo `index.html` em seu navegador ou use um servidor local:
```bash
# Usando Python
python -m http.server 8000

# Usando Node.js
npx serve .

# Usando PHP
php -S localhost:8000
```

3. Acesse `http://localhost:8000` no seu navegador
